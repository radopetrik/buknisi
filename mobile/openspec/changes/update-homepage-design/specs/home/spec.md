## ADDED Requirements
### Requirement: Home Screen Discovery
The Home screen SHALL provide users with search and discovery mechanisms for services.

#### Scenario: Search Entry
- **WHEN** the user views the Home screen
- **THEN** a search bar is displayed at the top allowing text input for "services or businesses".

#### Scenario: Service Categories
- **WHEN** the user views the Home screen
- **THEN** a list of service categories (e.g., Barber, Hair, Massage) is displayed as a horizontal scrollable rail.
- **AND** each category shows an icon/image and a label.

#### Scenario: Special Offers
- **WHEN** the user scrolls down
- **THEN** a "Special Offers" section is displayed with a horizontal list of businesses.
- **AND** each card shows the business image, rating, review count, name, and address.

#### Scenario: Recommended Businesses
- **WHEN** the user scrolls further
- **THEN** a "Recommended" section is displayed with a horizontal list of businesses.
- **AND** the presentation is similar to Special Offers.
