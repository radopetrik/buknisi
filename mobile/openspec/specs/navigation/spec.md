## Requirements
### Requirement: Main Navigation Menu
The application SHALL provide a main navigation menu (Tab Bar) always visible on top-level screens.

#### Scenario: Menu Items
- **WHEN** the user views the main menu
- **THEN** it must display the following 4 items in order:
  1. **Home** (Label: "Domov")
  2. **Explore** (Label: "Prehliadať")
  3. **Appointments** (Label: "Rezervácie")
  4. **Profile** (Label: "Profil")

#### Scenario: Default Screen
- **WHEN** the application starts
- **THEN** the **Home** screen is displayed by default.
